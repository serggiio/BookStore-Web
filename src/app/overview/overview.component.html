Book store CRUD

This is a simple bookstore with CRUD operations. This project was made the backend in .net Core 6 and in the frontend with Angular. The main entities are books, authors and categories. All these entities have a relational schema that is reflected in the data base.

BOOKS

One book can have multiple categories and authors.

AUTHORS

The author entitie is related to the books.

CATEGORIES

The categories are related with the books. in the book detail section you can add or delete these categories.

STACK

Some of the technologies used in this project are: .net Core, Angular, Dapper, SQL Server, Bootstrap, Fluent validator, Docker.
